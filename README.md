Responsive Netflix-inspired web application built for learning and practice. This project recreates the core look and feel of Netflix, including a modern UI, movie browsing experience, and responsive design.
Features
Clean and modern Netflix-style interface
Responsive design for desktop, tablet, and mobile
Hero banner section
Movie/show rows with posters
Hover effects and smooth UI interactions
Navigation bar with branding
Footer section
Fast and optimized frontend performance
Tech Stack
HTML
CSS
JavaScript
Purpose of This Project
This project was created to improve frontend development skills by recreating a popular streaming platform interface. It focuses on layout design, responsiveness, and interactive user experience.
